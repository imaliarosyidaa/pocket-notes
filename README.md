# Nama Website

Pocket Notes

Pocket Notes adalah sebuah website yang membantu produktivitas mahasiswa dengan menyediakan alat-alat untuk
mencatat catatan, pengingat tugas, manaejemen tugas dan lain sebagainya.

## Daftar Fitur Utama

### 1. Smart Notes

- Membuat catatan
- Edit & hapus catatan

### 2. Task Manager

- Mengubah status tugas

### 3. Study Planner

- Perencanaan topik belajar
- Pengingat waktu belajar
- Jadwal harian/mingguan

### 4. Quick Notula

- Fitur asisten cerdas untuk mencatat video youtube menjadi notulensi terstruktur secara instan.

### 5. Deep Search

- Menyajikan riset mendalam yang dirancang khusus untuk akademisi dan mahasiswa untuk menemukan referensi jurnal yang valid beserta ringkasanya
